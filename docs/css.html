Global CSS

Base uses relative sizing (rems, not pixels) for global typography. Sans-serif is the default font family and the global size is 16rem with a line height of 21rem.

Headings (h1)

The font size of headings can easily be configured at the top of the LESS / SASS file accordingly.

Second Heading (h2)

Along with standard headings, there are additional helper headings .h1 to .h6 which can style headings as alternative sizes but keep the semantics in play.

Third Heading (.h4 class applied)

An example of this is the above heading is a h3 with a .h4 class on it. This gives you the correct semantics of the h3 following the h2, but you get the look and feel of a h4.

Fourth Heading (.h3 class applied)

Now in this example, I have a h4 above with a .h3 class applied to it.

Fifth Heading (h5)

Overall, these helpers can be very handy when dealing with lots of individual components and headings.

Global Typography Helpers / Classes

Along with the heading helpers (classes), there are additional typography helpers too.

.inline-block Sets element to be inline block
.block Sets element to be block
.thin Applies a thin body weight (in base docs, the thin class is applied to most typography so you won't see any change on this example as the font is already thin).

.strong Applies a strong body weight
.heavy Applies a heavy body weight
.capitalize Transforms text to be capitalized
.uppercase Transforms text to uppercase
.em Gives an element emphasis

.center-text Aligns text to the center for all devices
.right-text Aligns text to the right for all devices
.tablet-left-text Aligns text to the left only for tablet devices
.tablet-center-text Aligns text to the center only for tablet devices
.tablet-right-text Aligns text to the right only for tablet devices
.mobile-left-text Aligns text to the left only for mobile devices
.mobile-center-text Aligns text to the center only for mobile devices
.mobile-right-text Aligns text to the right only for mobile devices

Definition List
The DL element should be used when you want incorporate a definition of a term in your document, it is often used in glossaries to define many terms, it is also used in “normal” documents when the author wishes to explain a term in a more detail (Like this definition).

</dl>

Forms and Buttons

The grid system can be used to add structure and layout to your forms.

Example form
What is your gender?
What hobbies do you enjoy? 
<!-- Form Elements -->
<form action="" method="get">
  <fieldset>
    <legend>Example form</legend>

      <!-- Input[type=text] -->
      <label for="name">Full Name</label>
      <input type="text" id="name" cols="60" placeholder="Your full name...">

      <!-- Selectbox -->
      <label for="age">Your Age</label>
      <select class="col" name="age" id="age">
        <optgroup label="Please choose your age">
        <option value="18">18</option>
        <option value="19">19</option>
        <option value="20">20</option>
        <option value="21">21</option>
        <option value="21+">21+</option>
      </select>

      <!-- Radio -->
      <strong class="normal block">What is your gender?</strong>
      <span class="checkbox">
        <input type="radio" name="gender" id="male">
        <label for="male">Male</label>
      </span>

      <!-- For multi-line checkboxes / radios, a parent .checkbox element must be wrapped -->
      <strong class="normal block">What hobbies do you enjoy?</strong>
      <span class="checkbox">
        <input type="checkbox" id="sports" name="sports">
        <label for="sports">Sports like Volleyball, Tennis, Table Tennis, Soccer and Football</label>
      </span>

      <!-- Textarea -->
      <label for="comments">Comments</label>
      <textarea name="comments" id="comments" cols="60" rows="7"></textarea>

      <!-- Reset Button -->
      <input type="reset" value="Reset Form" class="button grey-button">

      <!-- Submit Button -->
      <input type="submit" value="Send Form" class="button">

      <!-- Disabled Button -->
      <input type="disabled" value="Disabled Submit Button" disabled class="button">

      <!-- Unstyled Button -->
      <input type="submit" value="Unstyled Submit Button" class="unstyled-button">

  </fieldset>
</form>
